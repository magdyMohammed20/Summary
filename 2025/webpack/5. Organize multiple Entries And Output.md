(1) Organize multiple Entries And Output 
----------------------------------------
==> The Purpose Is If We Create Multiple Pages And Each Page Has It's Own Style , Script And Fonts Files

    webpack.config.js
    --------------------

    const path = require("path")

    module.exports = {
        ///////////// Here We Add Multiple Entries ///////////////////////
        entry: {
            index: "./index.js",
            explore: "./explore.js"
        },
        output: {
            ///////////// Here We Handle Multiple Output /////////////////////
            filename: "[name].bundle.js",
            path: path.resolve(__dirname, "dist"),
            assetModuleFilename: "asset/[hash][ext]",
            clean: true
        },
        module: {
            rules: [
                {
                    test: /.(png|jpe?g)$/,
                    type: "asset/resource"
                },
                {
                    test: /\.css$/,
                    use: ["style-loader" , "css-loader"]
                },
                {
                    test: /.scss$/i,
                    use: [
                        "style-loader",
                        "css-loader",
                        "sass-loader"
                    ],
                },
                {
                    test: /\.(ttf)$/,
                    type: "asset/resource"
                }
            ]
        }
    }



    index.html
    -------------

     <script src="./dist/index.bundle.js"></script>

    explore.html
    ---------------

    <script src="./dist/explore.bundle.js"></script>

(2) After Making Build For 'index.js' The Output Bundled File Will Called 'index.bundle.js'
    So If We Need To Rename It With Different Name We Can Change Config Of Entry File In 'webpack.config.js'
    And Don't For Get To Change Name Of Import In The Html File 

    webpack.config.js
    -----------------------
        
    entry: {
        index: {
            import: "./index.js",
            filename: "main-entry.js"
        },
        explore: "./explore.js"
    },

    index.html
    ---------------
        
    <script src="./dist/main-entry.js"></script>
