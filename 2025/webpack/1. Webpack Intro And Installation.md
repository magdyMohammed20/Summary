(3) Normal Usage If Setup
---------------------------
==> Create Basic Folder That Contains Html File And Js Files
==> Initialize The Package.json [npm init]
==> Install Webpack [npm i webpack webpack-cli --save-dev]
==> Create [webpack.config.js] File

    webpack.config.js
    -------------------

    const path = require("path")

    module.exports = {
        entry: "./index.js",
        output: {
            filename: "bundle.js",
            path: path.resolve(__dirname , "dist")
        }
    }

==> Change/Add 'build' Command In [package.json] File

    package.json
    --------------

    "scripts": {
        "build" : "webpack"
    },


==> Run [npm run build] For Generate The Minified Output


==> For Generate Non Minified Version Change The Build Attribute In [Package.json] To 

    {
        "build" : "webpack --mode development"
    }
