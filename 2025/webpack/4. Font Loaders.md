(1) Font Loaders
-------------------
==> For Include The Font File In Build Output
==> First Step : Is To Create 'fonts' Folder In Your Project And Add 'ttf' File Inside It
==> Second Step : In Sass File Or In Css File Use The Font In Your Style Nomrally
==> Third Step : Import The 'ttf' File In Js Script
==> Fourth Step: Add Font Loader In 'webpack.config.js'



    style/sassy.scss
    -----------------

    h1{
        font-family: "Dancing Script", cursive;
    }
    
    index.js
    -----------


    import png from "./assets/png.png"
    import './style/style.css'
    import './style/sassy.scss'
    import './fonts/DancingScript-VariableFont_wght.ttf' //////////// Import Here /////////////


    const btn = document.getElementById('my-btn')
    const logo = document.getElementById('logo')


    btn.addEventListener('click', () => {
    
        logo.src = png
        
    })


    webpack.config.js
    ---------------------

    module: {
            rules: [
                {
                    test: /.(png|jpe?g)$/,
                    type: "asset/resource"
                },
                {
                    test: /\.css$/,
                    use: ["style-loader" , "css-loader"]
                },
                {
                    test: /.scss$/i,
                    use: [
                        "style-loader",
                        "css-loader",
                        "sass-loader"
                    ],
                },

                ////////// This Is Loader ////////////////
                {
                    test: /\.(ttf)$/,
                    type: "asset/resource"
                }
            ]
        }
