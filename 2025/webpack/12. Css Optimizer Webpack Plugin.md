(1) Css Optimizer Webpack Plugin
-------------------------------------
==> This Plugin Uses Css Nano To Optimize And Minify The Css
==> Install: [npm i css-minimizer-webpack-plugin --save-dev]
==> Then Add It In 'webpack.prod.config.js' And After Run The Build Will Note That Css File In 'dist' Is Minified


    webpack.prod.config.js
    -------------------------

    const CssMinimizerPlugin = require('css-minimizer-webpack-plugin')

    module.exports = {
        optimization: {
            minimizer: [new CssMinimizerPlugin()]
        }
    }


(2) Optimize Other Html And Js Files 
--------------------------------------
==> When  optimization: {  minimizer: [new CssMinimizerPlugin()] } In Webpack config file 
    Webpack would only minify the CSS (because your custom minimizer list replaces the built-in one)

==> So To keeps the default JS minimizer You Should Add `...`

    webpack.prod.config.js
    -------------------------

    optimization: {
        minimizer: [`...` , new CssMinimizerPlugin()]
    },
