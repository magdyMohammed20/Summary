(1) Production & Development Modes
------------------------------------
==> In package.json File You Can Specify Commands Of Run Using Mode

      "scripts": {
            "build": "webpack --mode production",
            "dev": "webpack --mode ",
            "start" : "webpack serve --open --mode development"
        },


(2) Another Way For Dev And Prod Modes
----------------------------------------
==> You Can Split The 'webpack.config.js' File Into 2 Files 'webpack.dev.config.js' And 
    'webpack.prod.config.js' And Add 'mode' Prop For Define Each Mode For Each Config File

==> And 'devServer: { port: 3001 }' Remove It From The Production Config File

==> Update The Run Scripts For Each Config File


    webpack.dev.config.js
    -----------------------

    module.exports = {
        mode: "development",
        ....
    }


    webpack.prod.config.js
    -----------------------

    module.exports = {
        mode: "production",
        ....
    }

    package.json
    ---------------

      "scripts": {
            "build": "webpack --config webpack.prod.config.js",
            "dev": "webpack --config webpack.dev.config.js ",
            "start" : "webpack serve --open --mode development"
        },
