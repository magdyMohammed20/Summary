(1) Create Base Html Template
-------------------------------
==> Create It For Prevent Repeate The Code Like <head> , body , .... In Each Html File

    /templates/base.html
    -----------------------

    <!DOCTYPE html>
    <html lang="en">

    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="stylesheet" href="{{url_for('static' , filename='style.css')}}" />
        {% block head %} {% endblock %}
    </head>

    <body>
        {% block body%} {% endblock %}
    </body>

    </html>


    /templates/index.html (Note Here There Is No head or body As We Extends The Base Template)
    -------------------------
