0- Download Python Manager And Install It Then Run The Following Commands

1- Installation (.venv folder )
--------------------------------
==> python3 -m venv env

2- Activate the environment
----------------------------
==> env\Scripts\activate (In Powershell)

3- DeActivate the environment
-----------------------------
==> deactivate

4- Install Flask And Other Libs
----------------------------------
==> pip install Flask Flask-Scss Flask-SQLAlchemy

5- For Check The Installation 
--------------------------------
==> pip list

6- Generate requirements.txt
-----------------------------
==> pip freeze > requirements.txt 


7- File Structure
-------------------
==> Create [app.py] Manually : Initial File Of Flask
==> requirements.txt : Contains Third Party Libs That Used In The Project
==> Create [templates/] Manually : Contains The Actual Html Files
==> Create [static/] Manually : Contains Css Files Or Js Files


8- If Not Run From VScode Run "python app.py" In activation Mode Then Open The Port "http://127.0.0.1:5000/"
