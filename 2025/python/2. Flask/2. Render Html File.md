(1) Render Html File
-----------------------
==> In the Project Root File Create 'static' and 'templates' Folders And Inside The 'templaates' Create 'index.html'
    which is the home page for us

==> Inside 'app.py' Handle Render 'index.html' Like Following


    app.py
    --------

    from flask import Flask , render_template
    from flask_scss import Scss
    from flask_sqlalchemy import SQLAlchemy

    app = Flask(__name__)

    @app.route("/")
    def index():
        # For Here We Create 'static' Folder In Project root Folder And Create 'index.html' File
        return render_template("index.html")



    if __name__ in "__main__":
        app.run(debug=True)
