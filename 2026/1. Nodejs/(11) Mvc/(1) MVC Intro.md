(1) MVC (Module View Controller)
--------
==> Design Pattern Used In Software Development Primarily For Creating User Interfaces

(2) Components
-----------------
==> Model : Represents The Data , All Business Logic 
            Responsibile For Retrieving And Storing Data 
            Notify The View Of Any Changes In The Data

==> View : Represents User interface and represenation Of Data
           Displays The Data Provided By The Model
           Sends User Input To The Controller

==> Controller : Acts As Intermediary Between The Model And View
                 Processes User Input And updates The Model Accordingly
                 can also update the view to reflect changes in the model
