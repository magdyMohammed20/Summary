(1) REPL (Read Eval Print Loop)
---------------------------------
==> Interactive Programming Environment
==> In terminal if write 'node' Command Then Can Execute Any Code Or Mathematic Operation

(2) Js VS Node.Js
--------------------
==> Environment : Js Runs In Web Browsers
==> Execution : Js Execute In Browser Envronment
==> Threading : Js Is Single Threaded
==> Blocking : Js Can Block UI Thread
==> Global Object : In Js 'window'
==> Event Loop : Utilizes Browser Event Loop
==> Package Manager : n/a (Dependency management via npm)
==> Concurency Model : Based On Async Callbacks

==> Environment : NodeJs Is Server Side Runtime Environment
==> Execution : NodeJs Executes Outside Browser Environment
==> Threading : Nodejs Is Single Threaded But Supports Async i/o
==> Blocking : Nodejs Uses Non Blocking i/o operations
==> Global Object : In Nodejs 'global'
==> Event Loop : Utilizes NodeJs Event Loop
==> Package Manager : npm
==> Concurency Model : Based On Async Callbacks Supports Worker Threads , Clusters, ... etc

(3) In Node Js For Example Console Comes From 'global' object [global.console.log("Hellow World")]
