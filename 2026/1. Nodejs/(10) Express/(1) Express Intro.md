(1) Express.js
----------------
==> Minimalist Web Application Framework For Node.js
==> Provides set of features For Building Web And Mobile Apps
==> Designed To Be Lightweight , flexible and unopinionated (With No Architecture) allowing developers to create server side apps quickly and efficiently

(2) Express.js key features
-----------------------------
==> Routing : For Define Routes And Handle HTTP Requests
==> Middleware : Functions that have access to request and response objects and can modify/terminate the request/response cycle
==> template engines
==> Error Handling
==> Static File Serving
==> Restful Routing 
==> Session Management
==> Middleware Customization
