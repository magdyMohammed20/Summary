(1) Use Prettier In Tests
----------------------------

==> Add 'format' Command In Package.json scripts

    "scripts": {
        "format" : "prettier --ignore-path .gitignore --write \"**/*.{js,ts,jsx,tsx,scss,css}\""
    }

==> Run[yarn format]Or[npm run format]
