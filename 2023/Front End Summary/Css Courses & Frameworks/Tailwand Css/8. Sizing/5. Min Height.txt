(1) Min Height
-----------------
==> Utilities for setting the minimum height of an element

Class       Properties
--------------------------------
.min-h-0	min-height: 0;
.min-h-full	min-height: 100%;
.min-h-screen	min-height: 100vh;

(2) Samples
-------------

    <div class="bg-gray-600 min-h-screen"></div>


(3) Responsive Height
------------------------

    <div class="bg-gray-600 min-h-0 md:min-h-full lg:min-h-screen"></div>
