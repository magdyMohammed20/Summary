(1) Min-Width
---------------
==> Utilities for setting the minimum width of an element

Class           Properties
----------------------------------
.min-w-0	    min-width: 0;
.min-w-full	    min-width: 100%;


(2) Samples
-------------

  <div class="bg-gray-600">
    <div class="bg-gray-200 p-5 w-1/2 min-w-full">1</div>
  </div>


(3) Responsive Min-Width
--------------------------

    <div class="w-24 min-w-0 sm:min-w-full md:min-w-0 lg:min-w-full xl:min-w-0 ...">
    Target
    </div>
