(1) After creating cli app Must Change The Initial Bundle Id Or Project Id That Used In Process Of Publishing


(2) Steps To Change
---------------------
==> In Your App Search For '.project-name' (EX: .myApp) And Will Get All Occurences 
==> Then Can Change To Format Of (com.user-name.project-name) (EX: com.magdymohammed20.myApp)

(3) Change Initial 'PRODUCT_BUNDLE_IDENTIFIER' Value
------------------------------------------------
==> Search For 'PRODUCT_BUNDLE_IDENTIFIER' And Change Value Of It With The Bundle Id (EX: PRODUCT_BUNDLE_IDENTIFIER = "com.magdymohammed20.myApp")
