(1) Change Icon For ios
---------------------------
==> First Is To Upload Your App Icon On 'https://www.appicon.co/' And Will Download File That Contains 'AppIcon.appiconset' As Folder
Then Go To '/ios/app_name/Images.xcassets/AppIcon.appiconset' And Replace Folders


(2) Change Icon For Android
-------------------------------
==> First Is To Upload Your App Icon On 'https://www.appicon.co/' And Will Download File That Contains 'AppIcon.appiconset' As Folder 
    Then Go To 'android/app/src/main/res/mipmap-hdpi' And Replace Images That Contains With The New One
    But Consider Upload The Normal Icon And Rounded Icon
